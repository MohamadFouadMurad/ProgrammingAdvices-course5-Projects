# 🎮 C++ Logic & Console Games Portfolio

Welcome to my C++ games repository! This repository showcases a collection of interactive, terminal-based games designed to master algorithmic thinking, structural software architecture, and core logic handling in C++.

---

## 📋 Included Projects

### 1. Rock Paper Scissors
A dynamic console-based implementation of the classic game featuring a structural layout and interactive round tracking.
* **Core Concepts:** Custom structs for score states, safe typed enums for choices, and pseudo-random computer decisions.

### 2. Math Game
An educational, highly interactive arithmetic game designed to challenge users with mathematical operations across dynamic difficulty settings.
* **Core Concepts:** Dynamic difficulty calculation algorithms, continuous loop game states, and multi-conditional execution branches.

---

## 🛠️ Engineering Principles Applied
* **Structural Programming:** Completely modular design using pure, reusable, single-responsibility functions.
* **Input Boundary Validation:** Engineered strict data-validation loops to prevent user-entry bugs and runtime crashes.
* **State Management:** Utilizing clean structures to track and reset dynamic scores across active game loops.
